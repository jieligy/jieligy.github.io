Python爬虫之Requests库的基本用法

发表于 2018-04-25 | 分类于 Python |

什么是Requests库？

Requests库是用Python编写的，基于urllib，采用Apache2 Licensed 开源协议的HTTP库。相比urllib库，Requests库更加方便，可以节约我们大量的工作，完全满足HTTP测试需求。

参考：Requests库中文文档 ； 高级用法

安装

pip install request

Requests用法

实例

import requests
r = requests.get('http://www.baidu.com/')
print(r.status_code)  #请求状态码
print(r.url)          #获取请求的url
print(r.cookies)     #获取请求的cookie
print(r.content)     #获取二进制内容

r.status_code #响应状态码
r.raw #返回原始响应体，也就是 urllib 的 response 对象，使用 r.raw.read() 读取
r.content #字节方式的响应体，会自动为你解码 gzip 和 deflate 压缩
r.text #字符串方式的响应体，会自动根据响应头部的字符编码进行解码
r.headers #以字典对象存储服务器响应头，但是这个字典比较特殊，字典键不区分大小写，若键不存在则返回None

#特殊方法#
r.json() #Requests中内置的JSON解码器
r.raise_for_status() #失败请求(非200响应)抛出异常

运行结果：
200
http://www.baidu.com/

各种请求方式(HTTP测试网站：http://httpbin.org/)

import requests
requests.get('http://httpbin.org/')  #发送get请求
requests.post('http://httpbin.org/')  #发送post请求
requests.put('http://httpbin.org/')
requests.delete('http://httpbin.org/')

请求

基于GET请求

import requests
response = requests.get('http://httpbin.org/get')
print(response.text)

发送一个get请求，并打印返回的内容
还可以进行带参数的请求：

import requests
data = {'username':'jieli','password':'112233445566'}
response = requests.get('http://httpbin.org/get',params=data)
print(response.text)

把需要传入的参数数据生成一个字典然后调用params参数， 然后赋值即可。
post请求也是一样。

读取二进制

比如图片

import requests
r = requests.get('http://www.baidu.com/img/baidu_jgylogo3.gif')
print(r.content)    #content 是获取的图片页面源码，二进制数据
print(r.text)      #text 则是字符串代码

显然获取图片这种二进制数据，需要使用content方法
我们还可以保存图片

with open('logo3.gif'',wb') as f:
    f.write(r.content)
    print('下载成功！')

这样我们就保存了图片，在我们的文件夹中就可以看到。

解析json

import requests
import json
r = requests.get('http://httpbin.org/get')
print(type(r))
print(json.loads(r.text))
print(r.json())
print(type(r.json()))

运行结果

可以看出requests的 json 解析和 json的loads方法解析出来的都是同样的字典
所以说requests可以很方便的解析json数据。

headers

有时候我们访问一些网页，它会查看请求的对象是不是来源于浏览器，如果不是，它就会禁止我们爬虫的访问，所以，我们需要添加一个headers。
也就是user-agent信息。

import requests
data = {'username':'jieli','password':'1122334'}
headers = {'Mozilla/5.0 (Windows NT 6.1) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/66.0.3359.117 Safari/537.36'}
r = requests.get('http://httpbin.org/',data= data,headers=headers)
print(r.text)

基于post请求

我们只需要调用post方法就可以了，当然 form data是必然不可少的，传入一个data参数就可以了。
